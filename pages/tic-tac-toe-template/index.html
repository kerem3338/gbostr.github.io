Installing

Just download the zip file and extract it somewhere, then you can play the game using a web server, Thats all to run the game.

  • Easy Setup: Start playing the game by downloading a simple HTML file and running it on a web server.

  • Mobile Support: The game can be played smoothly on mobile devices.

  • Sound Effects: Click and win sounds are included for user-friendly experience.

  • Customizability: You can easily change the appearance and behavior of the game.

  • Lightness: Provides a fast and smooth gaming experience with minimal resource consumption.

  • Menu Skip: You can skip the menu and go directly to the main game screen upon request.

  • Version Tracking: The version number is specified to track which version of the game you are using.

  • Information Page: There is a special information page to get detailed information about the game.
